Program ini dibuat untuk membantu **diagnosa dini malnutrisi pada balita** dengan memanfaatkan metode **Forward Chaining** dan **Certainty Factor (CF)**.  
Aplikasi ini menerima input berupa **gejala-gejala klinis** yang dialami balita, kemudian sistem akan melakukan penalaran untuk menentukan kemungkinan jenis malnutrisi beserta tingkat kepastiannya.

- Menyediakan informasi tambahan mengenai tingkat kepastian diagnosa dengan metode **Certainty Factor**.

- Input gejala balita (misalnya: berat badan rendah, wajah pucat, rambut mudah rontok, dll).
- Proses inferensi menggunakan **Forward Chaining**.
- Perhitungan tingkat kepastian diagnosa dengan **Certainty Factor**.
- Output berupa kemungkinan jenis malnutrisi (misalnya: **Gizi Kurang, Gizi Buruk, Stunting, atau Normal**) beserta nilai CF.

## ğŸ§© Fitur Utama
- Input gejala balita (misalnya: berat badan rendah, wajah pucat, rambut mudah rontok, dll).
- Proses inferensi menggunakan **Forward Chaining**.
- Perhitungan tingkat kepastian diagnosa dengan **Certainty Factor**.
